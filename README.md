# WEBTRIS/ВЕБТРИС
A html & javascript based [Tetris](https://en.wikipedia.org/wiki/Tetris) clone
____________
## Modifications:
- Version 1.0

    - Color Theme ([Bubblegum Midnight](https://github.com/trzyglow/BubblegumMidnight-specsheet))

    - Adaptive Scaling

    - Fixed Music Loop
